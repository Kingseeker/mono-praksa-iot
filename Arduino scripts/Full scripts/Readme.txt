S ovim skriptama robot se pokrece preko web i desktop aplikacije.
Takoder, ove skripte ukljucuju prikupljanje i slanje GPS podataka,
te infracrvene senzore za detekciju prepreka.

Komande za pokretanje se mogu slati preko sucelja web ili desktop aplikacije.

Upload programa na odgovarajuci Arduino:
ArduinoControl - Arduino povezan sa prijenosnim ili stolnim racunalom
LoRaArduinoRobot - Arduino povezan na LoRa Shield, na robotu
MotorArduinoRobot - Arduino povezan s motorima, na robotu   

Nakon uploada, prvo treba prikljuciti Arduino s LoRa modulom na power bank, a zatim ukljuciti napajanje Arduino Duemillanove preko prekidaca.
